# Tati Nails - Сайт маникюрного мастера

## Описание
Сайт-портфолио для маникюрного мастера Tati Nails, представляющий услуги профессионального маникюра в Краснодаре.

## Используемые технологии

### Основные технологии
- **HTML5** - для структуры веб-страницы
- **CSS3** - для стилизации и анимаций
- **JavaScript (Vanilla JS)** - для интерактивности и функционала

### Библиотеки и шрифты
- **Google Fonts** - шрифты Playfair Display и Open Sans
- **Font Awesome 6** - иконки для интерфейса
- **CSS Variables** - для удобного управления стилями

### Особенности реализации
- Адаптивный дизайн (мобильная версия)
- Анимации при прокрутке
- Галерея работ с лайтбоксом
- Форма записи на прием с валидацией
- Плавная прокрутка по якорным ссылкам

### Оптимизация
- Оптимизированные изображения
- Минимизированные CSS и JavaScript
- Ленивая загрузка изображений lazy

## Установка и запуск
1. Клонируйте репозиторий
2. Откройте файл `index.html` в любом современном браузере

## Контакты
По вопросам сотрудничества и записи на прием обращайтесь через контактную форму на сайте.
